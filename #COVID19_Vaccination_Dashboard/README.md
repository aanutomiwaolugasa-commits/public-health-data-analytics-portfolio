# COVID-19 Global Vaccination Dashboard

## Project Overview

This Power BI dashboard analyzes global COVID-19 vaccination progress across 223 countries between 2020 and 2022.

## Objectives

- Analyze global vaccination progress
- Identify countries with the highest vaccination counts
- Visualize vaccination coverage across regions
- Examine vaccination trends from 2020–2022

## Tools Used

- Power BI
- DAX
- Microsoft Excel
- GitHub

## Key Insights

- China recorded the highest number of total vaccinations.
- India ranked second globally in total vaccinations.
- Global vaccination activity peaked around 2021 before declining in 2022.
- Vaccination coverage varied considerably across countries and regions.

## Future Improvements

Future versions of this dashboard will use the latest available vaccination record for each country to improve KPI accuracy and avoid double-counting cumulative vaccination totals across dates.

## Dataset

COVID-19 World Vaccination Progress Dataset
